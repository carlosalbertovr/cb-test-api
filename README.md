# API REST sencilla con TypeScript

## Endpoints

- `GET /` → responde texto plano `ok`
- `GET /api/info` → responde un JSON con información de prueba

## Ejecutar en local

```bash
npm install
npm run dev
```

Para producción:

```bash
npm run build
npm start
```

## Ejemplos

```bash
curl http://localhost:3000/
curl http://localhost:3000/api/info
```
